Rules

League Soccer

Rules :
  • All bets have to be placed before the game starts.
  • Bets are always for the end result of a game, including additional time and penalty shootout
Points :
  • 3 points if you bet the exact end result
  • 2 points if you bet the tendency of the end result
  • 1 point if you bet for the winner of the end result
