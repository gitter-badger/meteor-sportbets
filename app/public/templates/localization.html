Localization

Cool that you want to add a new language, that's awesome! We are using just-i18n for localizing Sportbets, what you can see below is the content of the current selected language.

 

To add a new language you have to :
  • Select your favorite, already existing source language (English is always the most complete one!)
  • Copy the json from below
  • Translate all values
  • Test your localization below (no worries, this will only change the language in your browser and will get reverted after a page refresh)
  • Send the json to max@hybris.com
