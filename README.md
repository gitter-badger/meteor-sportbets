# Introduction
This is a sport bets project based on the Meteor Framework!

# Quick Start 


### Install Meteor
- Visit http://docs.meteor.com/#quickstart

### Start Sportbets
- Goto Sportbets Root Folder and then to the app folder
- Type ``meteor``, hit enter

Easy, heh?
